# Liteshift - Host Server

## To install on your server
```bash
bash <(curl -s https://raw.githubusercontent.com/chethaslp/liteshift-host/refs/heads/main/install.sh)
```

## To build
```bash
npm run build
```

## To run (development)
```bash
npm run dev
```

## To run (production)
```bash
npm start
```
